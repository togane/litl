== Description ==
Leanex Lite is a minimalistic theme, which is designed for bloggers and creative folk. This theme has amazing customizing options: front-page sortable, choosing fonts, layout variations for portfolio page and blog page, any colors for overlay of the header image, and much more. This theme suitable for bloggers, photo-blogging, freelancer portfolio, creative agency website.
